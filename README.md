# Pet Shop Management System (VB Basic)

##  Overview

This project is a simple Pet Shop Management System developed using Visual Basic. It is designed to manage basic pet shop operations such as recording customer purchases and displaying order details.

##  Objective

* To understand basic programming concepts using Visual Basic
* To develop a simple system for handling pet shop transactions
* To practice user input and output handling

##  Features

* Input customer name
* Select how many pet
* Enter service and calculate total price
* Display order summary

##  What I Learned

* Basic programming logic (if-else, calculations)
* Event handling in Visual Basic
* Building a simple functional system

##  Limitations

* No database integration
* Data is not stored permanently
* Limited validation and features
